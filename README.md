# Developer Portfolio

A sleek, neon-accented developer portfolio built with **React, TypeScript, TailwindCSS, and Framer Motion**.  
This website showcases my experience, projects, and skills through a clean UI, subtle animations, and a cyber-professional design language.

---

## Features

### Modern Design
- Dark neon UI inspired by cyber-systems.
- Custom animated components (typewriter logs, geometric avatar, experience timeline).
- Smooth transitions powered by **Framer Motion**.

### Optimized Architecture
- Modular React components following clean TypeScript best practices.
- Reusable utilities and a scalable layout.
- Fully responsive across all devices.

### Experience and Education Timelines
- Interactive horizontal and vertical experience timelines.
- Soft hover animations and glowing UI nodes.

### Projects & Skills Sections
- Card-based layout for clean presentation.
- Tech stack badges and categorized skill display.

---

## Setup

### 1. Clone the repository
```
git clone https://github.com/nikolovaradica/developer-portfolio.git
cd developer-portfolio
```

### 2. Install dependencies
```
npm install
```

### 3. Run locally
```
npm run dev
```
